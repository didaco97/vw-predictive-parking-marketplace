# ParkPredict - UI/UX Design Outline

## Design System Foundation

### Color Palette
```
Primary Colors:
- VW Blue: #001E50 (Primary actions, headers)
- Electric Green: #00B0B9 (Accent, success states)
- Pure White: #FFFFFF (Backgrounds)
- Deep Gray: #1A1A1A (Text)

Secondary Colors:
- Light Blue: #E8F4F8 (Backgrounds, cards)
- Soft Gray: #F5F5F5 (Dividers, disabled states)
- Medium Gray: #757575 (Secondary text)

Status Colors:
- Success Green: #4CAF50 (Confirmed bookings)
- Warning Amber: #FFA726 (Low availability, expiring soon)
- Error Red: #EF5350 (Unavailable, errors)
- Info Blue: #42A5F5 (Tips, notifications)

Confidence Indicators:
- High (>80%): #4CAF50 (Green)
- Medium (60-80%): #FFA726 (Amber)
- Low (<60%): #EF5350 (Red)
```

### Typography
```
Headings:
- H1: 28px, Bold (Screen titles)
- H2: 24px, Semi-bold (Section headers)
- H3: 20px, Semi-bold (Card titles)
- H4: 18px, Medium (Subsections)

Body Text:
- Body Large: 16px, Regular (Primary content)
- Body Medium: 14px, Regular (Secondary content)
- Body Small: 12px, Regular (Labels, captions)

Special:
- Button Text: 16px, Semi-bold
- Price Text: 20px, Bold
- Confidence Score: 18px, Bold
```

### Spacing System
```
- XXS: 4px (Tight spacing)
- XS: 8px (Element spacing)
- S: 12px (Component spacing)
- M: 16px (Card padding)
- L: 24px (Section spacing)
- XL: 32px (Screen margins)
- XXL: 48px (Large sections)
```

### Icons
- Icon Set: Lucide Icons / Material Symbols
- Size: 20px (small), 24px (default), 32px (large)
- Style: Rounded, 2px stroke weight
